# 📚 Book Vibe
## 📖 Description

**Book Vibe** is a reading management web application that allows users to organize and track their book collections efficiently.

All books are displayed on a single page with essential information such as cover image, title, author, genre, and rating.
Clicking on a book navigates to a dedicated details page where users can view in-depth information, including reviews.

Users can categorize books into two lists:

- Read (already completed)
- Wishlist (want to read)

The listed books are displayed using tab-based navigation, and users can sort them based on page count and rating. All data is stored locally to ensure persistence.

## 🛠️ Technologies Used
- JavaScript (ES6+)
- React.js
- Tailwind CSS
- DaisyUI
- React Router
- Recharts
- React Icons
- Local Storage (for data persistence)
## ✨ Features
- 📚 Book Listing Page
- 🔍 Book Details Page
- 📌 Reading Status Management
- 📑 Tabbed List View
- 🔽 Sorting Functionality
- 💾 Local Storage Integration
- 📊 Data Visualization
- 🎨 Modern UI/UX
- 🧾 Summary

**Book Vibe** is a fully functional and user-friendly book management application designed to help users track and organize their reading habits.
It leverages modern frontend technologies to deliver a clean interface, smooth navigation, and reliable local data persistence.
